# athlete_injury_risk_assesment
What is This?
The last 4 weeks through March and April I attended multiple meetings for the Biotech Club over creating a machine learning model to determine if an athlete is injured or not. We explored the libraries in Python and how they read the data before we ultimately trained 2 models which were the decision tree model and the logistic regression model.

What did I learn?
We imported multiple libraries with functions that I learned to use in order to read the data in a provided csv file. The file contained different categories which the machine learned that some impact the injury risk assesment more than others.
